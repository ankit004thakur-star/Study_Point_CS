# Quick Start Guide

Get up and running with CS Study Material in just 5 minutes!

## ⚡ Quick Setup (30 seconds)

### Option 1: Direct Browser Opening
1. Download or clone the project
2. Double-click `index.html`
3. Start learning! 🎉

### Option 2: Local Server (Recommended)
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx serve

# Or any other local server
```
Then visit: `http://localhost:8000`

## 📚 First Steps

1. **Explore Topics**
   - Scroll to "Topics Covered" section
   - Click on any topic card to dive in

2. **Read Content**

   - Click any heading to jump to that section
   - Read code examples and practice problems

3. **Learn More**
   - Check the Resources section
   - External links lead to recommended materials
   - Practice with the suggested problems

## 🛠️ Troubleshooting

### Content Not Loading?
1. Refresh the page (Ctrl+R or Cmd+R)
2. Clear browser cache
3. Try a different browser
4. Check internet connection

### Styling Issues?
1. Refresh the page
2. Clear browser cache
3. Try without dark mode
4. Update your browser

### Can't Click on Topics?
1. Ensure JavaScript is enabled
2. Try a different browser
3. Clear cache and cookies
4. Report the issue
